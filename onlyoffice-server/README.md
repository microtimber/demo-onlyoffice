# Onlyoffice Server

基于 nest 的 Onlyoffice 示例。

> Onlyoffice@7.2 之后的版本默认开启了 JWT 加密，运行仓库例子时请先关闭 JWT 校验。


## 快速使用

```bash
# 安装依赖
pnpm install

# 开发
pnpm run dev

# 打包
pnpm run build
```
